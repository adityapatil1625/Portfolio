# Portfolio Website

A modern, responsive developer portfolio built with React, Vite, TypeScript, and Tailwind CSS. It showcases projects, skills, experience, and contact information with a clean, fast UI.

## Features

- Single-page portfolio layout
- Responsive design across devices
- Modular sections (Hero, About, Skills, Experience, Projects, Contact)
- UI components powered by shadcn/ui
- Fast builds and dev server via Vite

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Getting Started

### Prerequisites

- Node.js (LTS recommended)
- A package manager: npm, pnpm, yarn, or bun

### Installation

```sh
# Clone the repository
git clone https://github.com/adityapatil1625/Portfolio.git

# Go to the project directory
cd my_portfolio

# Install dependencies
npm install
```

### Run Locally

```sh
npm run dev
```

Then open the local URL shown in the terminal.

## Scripts

- `npm run dev` — start the development server
- `npm run build` — create a production build
- `npm run preview` — preview the production build locally
- `npm run lint` — run lint checks

## Customize Content

Edit the section components in:

- [src/components/sections](src/components/sections)

Shared UI components are located in:

- [src/components/ui](src/components/ui)

## Deployment

Build the project and deploy the `dist` folder to any static hosting provider (Vercel, Netlify, GitHub Pages, etc.).

```sh
npm run build
```
